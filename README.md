# Extension Chrome Poids Équipes FFT

Cette extension calcule le "poids" moyen d'une équipe de tennis et affiche les 4 meilleurs classements des joueurs à partir d'une page d'équipe sur le site `tenup.fft.fr`.

## Comment Installer

Comme il s'agit d'une extension non empaquetée, vous devez la charger manuellement dans Chrome.

1.  **Ouvrir la Page des Extensions Chrome**:
    *   Ouvrez Google Chrome.
    *   Accédez à `chrome://extensions`. Vous pouvez copier et coller ceci dans votre barre d'adresse.

2.  **Activer le Mode Développeur**:
    *   Dans le coin supérieur droit de la page des extensions, vous verrez un interrupteur pour le **"Mode développeur"**. Assurez-vous qu'il est **activé**.

3.  **Charger l'Extension**:
    *   Une nouvelle série de boutons apparaîtra. Cliquez sur le bouton **"Charger l'extension non empaquetée"**.
    *   Une boîte de dialogue de sélection de fichiers s'ouvrira. Accédez au répertoire où vous avez ces fichiers (le répertoire `poids-equipes`).
    *   Sélectionnez le dossier `poids-equipes` en entier et cliquez sur **"Sélectionner"**.

4.  **Prêt à Utiliser !**:
    *   L'extension "Poids Équipes FFT" devrait maintenant apparaître dans votre liste d'extensions.
    *   Vous verrez son icône de balle de tennis dans votre barre d'outils Chrome (vous devrez peut-être l'épingler en cliquant sur l'icône en forme de pièce de puzzle).

## Comment Utiliser

1.  Accédez à une page d'équipe sur le site `tenup.fft.fr`.
2.  Cliquez sur l'icône de la balle de tennis de l'extension dans votre barre d'outils Chrome.
3.  La popup affichera le poids de l'équipe calculé et les 4 meilleurs classements des joueurs de la page actuelle.
